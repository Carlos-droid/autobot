# Autonomous Swarm Researcher (ASR)
Self-improving agentic loop that builds and hardens Python code via local LLMs (Ollama).
- **Persistent Memory**: Cures context amnesia via JSON episodes.
- **AST Robustness**: Automated SRE/Fixer roles.
- **Semantic Feedback**: Learns from test failures.
- **Telemetry**: Real-time performance tracking.
